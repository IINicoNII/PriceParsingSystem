Описание: Система парсинга цен с интернет-магазинов (OZON) с уведомлениями в Telegram.
Технологии и навыки:

    Парсинг данных (BeautifulSoup/Scrapy).

    SQL (хранение истории цен).

    Telegram Bot (отправка уведомлений при снижении цены).

    Фоновые задачи (APScheduler/Celery для регулярного парсинга).

    REST API (Flask/Django для управления товарами).
    Что нового: Парсинг, работа с асинхронными задачами, планировщиками.
